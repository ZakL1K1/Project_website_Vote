**📊 Online Polls — Веб-приложение для создания и прохождения опросов**
**📌 Описание проекта**

Online Polls — это веб-приложение, разработанное на Flask, которое позволяет пользователям:

· регистрироваться и авторизоваться
· создавать опросы с вариантами ответов
· проходить опросы
· просматривать результаты в виде таблицы и графиков
· делиться ссылками на опросы

Проект реализует клиент-серверную архитектуру и использует SQLite в качестве базы данных.

**🚀 Основной функционал**
👤 Пользователи:
· Регистрация
· Авторизация
· Выход из системы
🗳 Опросы:
· Создание опросов
· Добавление вариантов ответов
· Просмотр списка своих опросов
· Публичный доступ к опросам по ссылке
📊 Голосование:
· Выбор варианта ответа
· Защита от повторного голосования
· Подсчёт результатов
📈 Результаты:
· Отображение результатов голосования
· Визуализация данных с помощью графиков (Chart.js)
**🛠 Используемые технологии**
· Python 3
· Flask
· SQLite3
· HTML5
· CSS3
· Jinja2 Templates
· Chart.js (графики)
